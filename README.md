# Education Platform

## Overview
분산된 교육 서비스를 하나의 플랫폼으로 통합한 시스템입니다.
상품, 결제, 학습관리, 사용자 관리 등 다양한 기능을 포함하고 있습니다.

## My Role
- 통합 플랫폼 구축 프로젝트 참여
- 핵심 기능 개발 및 운영
- REST API 설계 및 개발
- 외부 시스템 연동 구현
- 성능 개선 및 장애 대응

## Tech Stack
- PHP (CodeIgniter, Laravel)
- MySQL
- REST API

## Key Features
- 상품 및 결제 시스템
- 학습 관리 시스템 (LMS)
- 사용자 관리 및 이벤트 기능
- REST API 제공

## External Integration
- SMS
- 카카오톡
- PG 결제 시스템

## Challenges & Solutions
서비스 운영 중 데이터 증가로 인한 성능 저하가 발생하여
DB 인덱스 최적화와 쿼리 튜닝을 통해 응답 속도를 개선하였습니다.

## Improvements
캐싱 구조나 추가적인 분리 설계를 통해 성능 확장성을 높일 수 있습니다.
